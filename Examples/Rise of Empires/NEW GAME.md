## Instructions for Agent

When the player invokes this file:

1. Read all files in `agent/` to understand your role.
2. Read all files in `game/` to understand the game definition.
3. Read `settings/custom.json` for active configuration.
4. Detect the player's language preference (ask if unclear: English or 中文).
5. Ask the player for a **save name** (unique identifier for this playthrough).
6. Execute the initialization procedure in `game/init.md`.
7. Begin the game session.
